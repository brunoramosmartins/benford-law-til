# Data

Bundled datasets used by the empirical and fraud-detection chapters of the TIL.

## Layout

```
data/
├── raw/                    # Source data, committed for reproducibility
│   └── world_cities.csv    # Generated by scripts/build_datasets.py
└── README.md               # this file
```

## How to populate `data/raw/`

The CSV files in `data/raw/` are reproduced from public sources by a build
script, not shipped pre-baked. To populate them:

```bash
python scripts/build_datasets.py
```

This downloads the **SimpleMaps World Cities (Basic)** dataset
(<https://simplemaps.com/data/world-cities>, CC BY 4.0) and writes a trimmed
CSV to `data/raw/world_cities.csv`. After running once, commit the resulting
file:

```bash
git add data/raw/world_cities.csv
git commit -m "chore(data): bundle SimpleMaps world-cities snapshot"
```
